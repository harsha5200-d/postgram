## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas Account
- ImageKit.io Account

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/postgram.git
   cd postgram
   ```

2. **Setup Backend**
   ```bash
   cd BACKEND
   npm install
   ```
   Create a `.env` file in the `BACKEND` directory:
   ```env
   IMAGE_PUBLIC_KEY = your_key
   IMAGE_PRIVATE_KEY = your_key
   IMAGE_URL_ENDPOINT = your_endpoint
   MONGO_URI = your_mongodb_uri
   ```

3. **Setup Frontend**
   ```bash
   cd ../frontend/user_interface
   npm install
   ```

4. **Run Locally**
   Return to the root directory and run:
   ```bash
   npm install
   npm run dev
   ```

## 🎨 UI Design Philosophy

PostGram uses a **Glassmorphic Design System**:
- **Transparency**: Subtle background blurs for depth.
- **Vibrant Gradients**: Deep slate and sky blue palettes.
- **Typography**: Utilizing 'Inter' for maximum readability.
- **Micro-animations**: Smooth hover transitions and loading states.
